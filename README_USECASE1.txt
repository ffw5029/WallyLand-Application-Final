Wallyland Use Case #1: Rides Use Case
===================================================================================================================
Included in this file is the rides use case. Upon running the application follow these steps: 

1. In the Main Menu, click the top button which says "Ride" 

2. Select a ride from "Phantom", "Black Widow", and "Bizarro" 

3. View each respective ride's details and line/capacity information

4. Click Back Buttons to return to the Ride Menu to view the other rides

===================================================================================================================
